# 材质编辑器

## 说明

以 Shader 为维度统一管理工程内所有材质，支持：

- 按 **Keyword** 筛选材质
- 按 **属性值** 筛选材质
- 批量 **设置属性**（Float / Vector / Color / Texture）
- 批量 **设置 Keyword 开关**
- 一键 **清除所有无效 Keyword**

## 使用方式

1. 打开窗口：菜单 `SpaceTime/Tool/材质编辑器`
2. 点击 **刷新 Shader 列表** 扫描工程内全部材质
3. 在 Shader 列表中点击目标 Shader，右侧按分组展示该 Shader 下所有材质
4. 根据需求选择对应操作（见下方操作说明）

## 界面布局

```
┌─────────────────────────────────────────────────────┐
│  [刷新 Shader 列表]                                  │  ← 顶部工具栏
├──────────────────────────────┬──────────────────────┤
│  Shader 列表（横向滚动）      │  分组索引（可点击）   │  ← 中间区域
├──────────────────────────────┴──────────────────────┤
│  [剔除无效 Keyword]                                  │
│  Keyword 筛选行                                      │  ← 底部工具栏
│  属性筛选 & 批量设置行                               │
│  Keyword 批量设置行                                  │
├─────────────────────────────────────────────────────┤
│  当前分组材质列表（点击高亮并 Ping 到 Project 窗口） │  ← 材质列表
└─────────────────────────────────────────────────────┘
```

## 操作说明

### 刷新 Shader 列表

点击顶部 **刷新 Shader 列表** 按钮，扫描工程内所有 `.mat` 文件，并以 Shader 分组展示。

### 按 Keyword 筛选材质

- 在 **筛选的 Keyword 列表** 输入框填入一个或多个 Keyword（空格分隔）
- 点击 **根据 Keyword 列表筛选材质**，列表刷新为包含所有指定 Keyword 的材质

### 按属性值筛选材质

- 填写 **材质属性名字**、选择 **材质属性类型**、填写 **材质属性值**
- 点击 **根据材质属性筛选材质**，筛选出属性值匹配的材质

### 批量设置属性

填写属性名、类型和值后，点击 **设置所有材质属性** 对当前 Shader 下全部材质生效。

> Vector / Color 类型的属性值格式：四个分量以 **英文逗号** 分隔，例如 `1,0,0,1`

### 批量设置 Keyword

填写 **材质 Keyword 名字** 和 **开关值**（`> 0` 表示开启），点击 **设置所有材质 Keyword**。

### 清除无效 Keyword

点击 **剔除所有的无效 Keyword**，对当前分组所有材质清理未在 Shader 中声明的 Keyword。

## 枚举类型说明

| 枚举 `MaterialSearchModifyMatType` | 含义 |
|---|---|
| Float | 浮点属性 |
| Vector | 四维向量属性 |
| Color | 颜色属性 |
| Texture | 贴图属性 |

## 注意事项

- 操作会直接修改材质资产，建议操作前备份工程
- Keyword 筛选多个时取 **与（AND）** 关系，即材质需同时包含所有指定 Keyword
- 分组索引从 1 开始显示，每组最多包含若干材质（由内部分组逻辑决定）
